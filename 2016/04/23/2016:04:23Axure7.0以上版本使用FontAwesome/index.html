Axure使用 Font Awesome图标

2016-04-23
Tags: ［原型设计］.
Categories: Design##分类.

1

Axure需要是7.0以上版本。因为最早接触的前端框架是 bootstrap，所以对 Font Awesome一直有一种情感。虽然个人觉得 Font Awesome的图标设计偏圆润，感觉略显棱角不足，不是非常好看。

在 Font Awesome官网下载字体包 传送门,安装 Font Awesome

2

重启 Axure，自动从字体库更新 Font Awesome字体

3

点击发布,选择预览选项

4

点击配置

5

选择Web字体，如图，添加 Font Awesome css 链接

对应的uri 为 http://netdna.bootstrapcdn.com/font-awesome/3.2.1/css/font-awesome.css

6

下载安装 Font Awesome Widget Library，传送门

注：Axure Widget Library安装可自行百度，简言之，将下载好的 rplib加入 Axure文件夹下的 Library目录，在 Axure中选择载入元件库即可

7

此时 已经可以在 Font Awesome已经可以在 Axure中正常拖拽使用了，需要注意的是，Font Awesome是一种字体，在未安装 Font Awesome字体的电脑上无法显示，可以把自己使用的Font Awesome图标转化成图片，这样就可以避免该问题，具体见下图
